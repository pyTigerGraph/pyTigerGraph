    Contribute to pyTigerGraph

    It is easy! Just follow the steps below: - Fork the Original Repo - Clone your forked Version - Make additions, modifications - Note: good practice to do a git fetch upstream to make sure both repos are in sync - Push to your forked Repo - Issue Pull Request to the original repository
